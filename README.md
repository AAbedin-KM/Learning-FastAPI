# what is FastAPI:
FastAPI is a modern Python web framework for building APIs quickly with automatic validation, documentation, and type hints. It's extremely fast (hence the name), uses async/await for handling concurrent requests, and automatically generates interactive API docs (Swagger UI) from your code.

# What will you learn:
- Building REST APIs with path, query, and request body parameters
- API documentation and validation with Pydantic schemas
- Database integration (models, relationships, CRUD operations)
- User authentication with JWT tokens
- Error handling and status codes
- API routing and repository patterns
- Deployment strategies
